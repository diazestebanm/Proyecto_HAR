Descripción detallada

HAL de persistencia del Nodo 2 con backend serial hacia el host.

Nota
El nombre del modulo se conserva para no romper la arquitectura ya existente, pero actualmente no monta una microSD. En esta version la persistencia real se hace por puerto serie y un script Python arma el CSV definitivo en el computador.
