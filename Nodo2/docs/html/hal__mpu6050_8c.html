Descripción detallada

Implementacion de la HAL del MPU6050 para el Nodo 2.

Este modulo se encarga de detectar el sensor, configurarlo en bajo consumo, habilitar la interrupcion de data-ready, calibrar el sesgo del giroscopio y capturar ventanas locales listas para la etapa de extraccion de rasgos.
