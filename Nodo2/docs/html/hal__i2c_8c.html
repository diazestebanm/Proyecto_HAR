Descripción detallada

Implementacion de la HAL I2C del Nodo 2.

Centraliza la inicializacion, cierre y transferencias basicas del bus I2C usado por el MPU6050. La meta es ofrecer una capa pequena y estable para que la parte de sensores no repita configuraciones del driver.

Lee uno o mas bytes consecutivos desde un registro del esclavo.

Lee uno o mas bytes a partir de un registro de un esclavo I2C.

La transaccion se hace con una fase de escritura del registro seguida de un repeated-start para la lectura, que es el patron esperado por el MPU6050.

Parámetros
[in]dev_addrDireccion del esclavo.
[in]regRegistro inicial.
[out]dataBuffer destino.
[in]lenCantidad de bytes a leer.
