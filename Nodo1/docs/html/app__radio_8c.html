Descripción detallada

Implementa la capa de radio del Nodo 1 basada en Wi-Fi STA y ESP-NOW.

Este modulo prepara la red, registra el callback de confirmacion de envio y ofrece una API bloqueante para despachar el payload de actividad construido por la FSM principal.
